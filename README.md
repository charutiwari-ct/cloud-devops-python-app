# Cloud DevOps Python App

A simple beginner-friendly project built to demonstrate core cloud and DevOps concepts using Python, Docker, GitHub Actions, and AWS EC2.

## Project Goal
This project is designed to showcase skills relevant to a Junior Cloud DevOps Engineer role, including:
- Python scripting
- Git and version control
- Docker containerization
- CI/CD with GitHub Actions
- Basic AWS EC2 deployment

## Planned Features
- Flask-based Python application
- `/` endpoint
- `/health` endpoint
- `/time` endpoint
- Automated tests with pytest
- Dockerized application
- CI pipeline with GitHub Actions
- Deployment to AWS EC2

## Tech Stack
- Python
- Flask
- Pytest
- Docker
- GitHub Actions
- AWS EC2
